# Guia de Contribuição

Obrigado por considerar contribuir para este projeto!

## Como Contribuir

### 1. Reportando Bugs
- Use a seção de Issues do GitHub
- Inclua detalhes sobre o ambiente
- Adicione passos para reproduzir o bug
- Inclua screenshots se possível

### 2. Sugerindo Melhorias
- Descreva claramente a melhoria
- Explique por que seria útil
- Sugira uma possível implementação

### 3. Enviando Pull Requests
1. Fork o repositório
2. Crie uma branch para sua feature
3. Adicione testes se aplicável
4. Atualize a documentação
5. Envie o Pull Request

## Padrões de Código
- Use docstrings para documentação
- Siga PEP 8 para estilo Python
- Adicione type hints quando possível
- Mantenha funções pequenas e focadas
